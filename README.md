### Pre-requisitos 📋

- [ ] Node
- [ ] NPM

### Instalación 🔧

Instala el proyecto por medio de Git Clone o descargando el archivo Zip.

Una vez instalado no olvides enejutar:

```
npm install
```

## Despliegue 📦

Ejecuta el proyecto con:

```
npm start
```
